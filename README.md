✅ Todo App

A simple and efficient Todo Application built using React, HTML, CSS, and JavaScript to help users manage daily tasks with ease.

🚀 Features
➕ Add new tasks
❌ Delete tasks
✔️ Mark tasks as completed
📝 Edit existing tasks
💾 Data persistence (Local Storage)
📱 Responsive design

🛠️ Tech Stack
React.js – Component-based UI
HTML5 – Structure
CSS3 – Styling
JavaScript (ES6+) – Logic & functionality

🎯 How It Works
Users can add tasks through an input field
Tasks are stored in local storage for persistence
Users can mark tasks as completed or delete them
React updates UI dynamically without page reload
